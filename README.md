# DebateAI - AI-Powered Debate Practice Platform

## Project Overview

DebateAI is an interactive web application that helps users practice and improve their debating skills by engaging with an advanced AI opponent. Master the art of argumentation, critical thinking, and persuasive communication.

## Getting Started

### Prerequisites

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Local Development

Follow these steps to run the project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Technologies Used

This project is built with modern web technologies:

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn/ui** - High-quality UI components
- **Tailwind CSS** - Utility-first CSS framework
- **React Router** - Client-side routing
- **React Query** - Data fetching and state management

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Features

- **Interactive AI-powered debate sessions** - Practice debating with advanced AI opponents
- **Multiple debate topics** - Choose from 20+ curated topics across various categories
- **Real-time scoring and feedback** - Get instant analysis of your arguments
- **Debate History** - Track your progress and review past debates
- **Global Leaderboard** - Compete with other debaters worldwide
- **User Profile & Settings** - Customize your experience and preferences
- **Responsive design** - Works seamlessly on all devices
- **Modern, intuitive UI** - Built with shadcn/ui components
